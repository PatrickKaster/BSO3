getting BSO3

:: where to get and how to integrate into GroIMP? ::

You may download a "compiled" Java bytecode version of the plugin here. Unzip the contents of the file to your GroIMP plugin directory! Upon starting your GroIMP installation, you may test the installation by trying out the demo from the walkthrough.

Or you may download the source distribution of the BSO3 plugin from it's GitHub repository. If you are building GroIMP from source using an IDE like eclipse, each module is a single project. So just a new project from existing sources for the BSO3-plugin. You may find additional information about how to build GroIMP from source on the GroIMP-site.
