acknowledgement

:: saying thanks to the giants on whose shoulder I stand ::

I would like to thank Sander van Rossen and Matthew Baranowski for writing their article and making their code available to the public. I especially thank Sander van Rossen for getting in touch with me on short notice via his blog, when I ran into problems and he confirmed the error where I assumed it to be.

Furthermore I would like to thank Florian Schöler who did a significant part of the job, especially all the binding of the algorithm to the GroIMP system were puzzled out by him. In this context I would like to mention that this plugin is a small part of his Ph.D. with the CROP.SENSe.net project. So if anybody looked into the code and wondered where the Java domain of the plugin originates from, now you know.

Last but not least I would like to acknowledge the tremendous work Ole Kniemeyer did with creating the GroIMP-modeller.
